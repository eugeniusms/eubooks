## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

- Home        : http://localhost:3000/ 
- Login       : http://localhost:3000/login
- Book Search : http://localhost:3000/books
- About       : http://localhost:3000/about
